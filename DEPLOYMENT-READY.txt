🚀 DÉPLOIEMENT EN 3 ÉTAPES (VPS Linux + Docker)

ÉTAPE 1 : Préparer votre .env
─────────────────────────────
  $ cp .env.example .env
  $ nano .env

  Remplacer    : LLM_PROVIDER=openai
                 OPENAI_API_KEY=sk-votre-clé-complète

ÉTAPE 2 : Construire l'image Docker
────────────────────────────────────
  $ docker-compose build

ÉTAPE 3 : Lancer l'application
───────────────────────────────
  $ docker-compose up -d

  ✓ L'App est accessible à → http://votre-serveur:8000
  ✓ API Documentation     → http://votre-serveur:8000/docs
  ✓ Health Check          → http://votre-serveur:8000/health

═══════════════════════════════════════════════════════════════

📋 COMMANDES UTILES

Voir les logs en temps réel :
  $ docker-compose logs -f

Arrêter l'application :
  $ docker-compose down

Redémarrer :
  $ docker-compose restart

Mettre à jour (depuis GitHub) :
  $ git pull origin main
  $ docker-compose build --no-cache
  $ docker-compose up -d

═══════════════════════════════════════════════════════════════

🔒 BONUS : HTTPS gratuit avec Let's Encrypt

Si vous avez un domaine, installer Nginx + Certbot :

  $ sudo apt install -y nginx certbot python3-certbot-nginx
  
  # Créer le proxy Nginx
  $ sudo nano /etc/nginx/sites-available/openmapagents
  
  # Copier la config depuis QUICKSTART-DEPLOY.md
  
  # Obtenir le certificat SSL
  $ sudo certbot --nginx -d votre-domaine.com

   Votre app sera accessible à → https://votre-domaine.com
