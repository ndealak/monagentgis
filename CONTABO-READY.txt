🚀 DÉPLOIEMENT EN 3 COMMANDES

Étape 1 : Se connecter
─────────────────────
  ssh root@votre-ip-contabo

Étape 2 : Installer automatiquement
────────────────────────────────────
  git clone https://github.com/diouck/openmapagents.git
  cd openmapagents
  ./scripts/install-contabo.sh

Étape 3 : Configurer et lancer
───────────────────────────────
  cp .env.example .env
  nano .env  # Ajouter votre clé OpenAI
  docker-compose build
  docker-compose up -d

✅ Application accessible à : http://votre-ip-contabo:8000

💡 COMMANDES UTILES APRÈS DÉPLOIEMENT

Voir les logs en direct :
  cd ~/openmapagents && docker-compose logs -f

Redémarrer l'app :
  docker-compose restart

Arrêter l'app :
  docker-compose down

Reprendre après arrêt :
  docker-compose up -d

Voir l'état :
  docker-compose ps

Vérifier les ressources :
  free -h && nproc && df -h

Accéder aux logs Nginx :
  sudo tail -f /var/log/nginx/openmapagents-error.log

🔒 RECOMMANDATIONS SÉCURITÉ

□ Changer le mot de passe root
□ Ajouter une clé SSH (à la place du mot de passe)
□ Configurer un firewall :
    sudo ufw enable
    sudo ufw allow 22/tcp
    sudo ufw allow 80/tcp
    sudo ufw allow 443/tcp

□ Mettre à jour régulièrement :
    sudo apt update && sudo apt upgrade -y

□ Activer SSL automatique (Let's Encrypt)
    ./scripts/setup-nginx.sh votre-domaine.com
